# Imp Counter

Документация: [русский](README.ru.md) | [English](README.en.md)

Автор: Impullsss

Небольшой system-agnostic module для Foundry VTT. Он добавляет два общих счётчика, которые удобно использовать для Advantage, Group Advantage или любого другого ресурса стола.

Технический module id: `imp-counter`.

## Что умеет

- Открывает два маленьких resizable windows.
- Окна можно двигать за строку с названием и сужать наведением на край окна, без видимого resize handle.
- В строке с названием есть компактная белая close control.
- Внизу слева есть кнопка `Adv`, которая открывает или скрывает оба окна.
- Launcher плавно поднимается выше Foundry player list, чтобы не перекрывать игроков.
- Рядом с launcher counters есть кнопка `Обновить`: она сбрасывает доступные счётчики в 0 и меняется на `Вернуть`, если можно откатить это действие.
- Нажатие на первый или второй launcher-counter открывает соответствующее окно, а повторное нажатие скрывает его.
- Значения общие для всего world и синхронизируются через `game.settings`.
- GM всегда может менять оба счётчика.
- В module settings можно разрешить или запретить игрокам менять первое и второе окно отдельно.
- Значения меняются кнопками `+` и `-`; `Shift-click` меняет сразу на 5.
- Значение можно вписать вручную в поле.
- Текст настроек и интерфейса берётся из `lang/ru.json` или `lang/en.json`, в зависимости от языка Foundry.

## Settings

- `Текст первого окна` - название первого счётчика.
- `Текст второго окна` - название второго счётчика.
- `Игроки могут менять первое окно` - разрешение для игроков.
- `Игроки могут менять второе окно` - разрешение для игроков.
- `Разрешить отрицательные значения` - по умолчанию выключено, потому что Advantage обычно не уходит ниже 0.
- `Сбрасывать при создании Combat` - опциональный auto-reset в 0 при создании нового Combat.
- `Открывать окна при входе` - client setting для текущего пользователя. По умолчанию выключено.
- `Показывать кнопку Adv` - client setting для текущего пользователя.

## Для игроков

Если игроку разрешено менять окно, его `+`, `-` и ручной ввод отправляют request активному GM через module socket. Если активного GM нет, значение не сохранится и появится warning.

## Console API

Модуль также оставляет маленький API:

```js
game.impCounter.open();           // открыть оба окна
game.impCounter.open("one");      // открыть первое окно
game.impCounter.open("two");      // открыть второе окно
game.impCounter.close();          // закрыть оба окна
game.impCounter.set("one", 3);    // поставить значение
game.impCounter.adjust("two", 1); // изменить значение
```

## Идеи для следующей версии

- Цветовые thresholds: например 0 серый, 1-3 зелёный, 4+ красный.
- Third counter mode для игр, где нужно больше двух сторон.
- Scene-specific values вместо одного общего world value.
- Chat log при изменении Advantage.
